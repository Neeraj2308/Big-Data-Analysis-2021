\paragraph{Calling value of pi}\label{calling-value-of-pi}}

To get the value of pi use math library in python. The value of the pi
can be called using \texttt{math.pi}. Donot use \texttt{math.pi()}. Pi
value is stored in variable name \texttt{pi} in math variable. Using
bracket after pi means you are calling pi as a function.

\paragraph{Practice: Normal Density}\label{practice-normal-density}}

Define function for normal density, that takes 3 inputs
\texttt{x,\ mu,\ and\ sigma}. The formula for the normal density is:
\[f(x; \mu, \sigma) =\frac{1}{\sigma \sqrt{2\pi}} exp\left[ -\frac{1}{2} \left( \frac{x - \mu}{\sigma}\right)^2 \right]  \;\;\;\;\;\;\;\;\;\sigma > 0 \tag{1}\]

    \begin{tcolorbox}[breakable, size=fbox, boxrule=1pt, pad at break*=1mm,colback=cellbackground, colframe=cellborder]

    \PY{k}{if} \PY{n}{sigma} \PY{o}{\PYZlt{}}\PY{o}{=} \PY{l+m+mi}{0}\PY{p}{:}
        \PY{k}{return}\PY{p}{(}\PY{l+s+s2}{\PYZdq{}}\PY{l+s+s2}{Sigma should be positive}\PY{l+s+s2}{\PYZdq{}}\PY{p}{)}
    \PY{n}{a} \PY{o}{=} \PY{l+m+mi}{1} \PY{o}{/} \PY{p}{(}\PY{n}{sigma} \PY{o}{*} \PY{p}{(}\PY{l+m+mi}{2} \PY{o}{*} \PY{n}{math}\PY{o}{.}\PY{n}{pi}\PY{p}{)}\PY{o}{*}\PY{o}{*}\PY{l+m+mf}{0.5}\PY{p}{)}
    \PY{n}{b} \PY{o}{=} \PY{o}{\PYZhy{}}\PY{l+m+mf}{0.5} \PY{o}{*} \PY{p}{(}\PY{p}{(}\PY{n}{x} \PY{o}{\PYZhy{}} \PY{n}{mu}\PY{p}{)} \PY{o}{/} \PY{n}{sigma}\PY{p}{)}\PY{o}{*}\PY{o}{*}\PY{l+m+mi}{2}
